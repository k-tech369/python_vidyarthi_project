Project Statement – MBTI Personality Profiler

## Problem Statement

Personality influences how people think, behave, communicate, and make decisions.
However, many individuals do not have a simple way to understand their personality traits scientifically.

The MBTI Personality Profiler solves this by providing:

A structured questionnaire

Automated scoring

A clear explanation of personality type
This helps users gain self-awareness and understand their personal strengths, communication style, and decision-making habits.


## Scope of the Project 

The scope of this project includes:

#**Core Functionalities**

Display MBTI-based questions

Take user inputs and validate answers

Calculate scores for all four dimensions:

Extraversion (E) vs Introversion (I)

Sensing (S) vs Intuition (N)

Thinking (T) vs Feeling (F)

Judging (J) vs Perceiving (P)


Generate final MBTI type (e.g., INFP, ESTJ)

Display explanation for the personality type


**#Secondary Scope
**
Modular code organization

Clean program flow with functions

Easy extensibility (add more questions or logic)

Optional diagrams:

Use Case Diagram

Workflow Diagram


## Target Users

The MBTI Personality Profiler is designed for:

Students – to understand learning and communication styles

Professionals – for self-evaluation and growth

HR teams – for pre-assessment or training workshops

Psychology beginners – to learn personality models

General users – who want to understand themselves better


The tool is beginner-friendly and requires no technical knowledge.



## High-Level Features

1.⁠ ⁠User-Friendly Interface

Simple CLI-based interaction makes the test easy to attempt.

2.⁠ ⁠Question Bank

A structured set of MBTI-related questions.

3.⁠ ⁠Automated Scoring System

Each response contributes to four personality dimensions.

4.⁠ ⁠Type Generator

The system calculates and returns the final 4-letter MBTI type.

5.⁠ ⁠Personality Explanation

After generating the type, the tool provides:

Strengths

Overview

Behavioral tendencies


6.⁠ ⁠Easy Customization

Developers can:

Add more questions

Modify scoring rules

Edit explanations
